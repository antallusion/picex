# Image Extractor

## Описание
Этот проект позволяет извлекать изображения с указанных веб-сайтов, используя браузер Puppeteer и предоставляя интерфейс для загрузки изображений.

## Структура проекта
Проект разделен на две части: 
- **Backend**: Серверная часть на Node.js с использованием Puppeteer для парсинга изображений.
- **Frontend**: Клиентская часть на React, предоставляющая интерфейс для ввода URL и отображения извлеченных изображений.
